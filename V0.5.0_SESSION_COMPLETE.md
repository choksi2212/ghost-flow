## ðŸ”„ Integration Examples

### WASM in React
```jsx
import { useEffect, useState } from 'react';
import init, { WasmTensor } from 'ghostflow-wasm';

function MLComponent() {
    const [result, setResult] = useState(null);
    
    useEffect(() => {
        init().then(() => {
            const tensor = WasmTensor.random([2, 2]);
            setResult(tensor.data());
        });
    }, []);
    
    return <div>Result: {JSON.stringify(result)}</div>;
}
```

### FFI in Python
```python
from ctypes import *

lib = CDLL('./libghostflow_ffi.so')
lib.ghostflow_init()

data = (c_float * 4)(1.0, 2.0, 3.0, 4.0)
shape = (c_size_t * 2)(2, 2)
tensor = c_void_p()

lib.ghostflow_tensor_create(data, 4, shape, 2, byref(tensor))
```

### REST API in Node.js
```javascript
const axios = require('axios');

async function predict(modelId, input) {
    const response = await axios.post(
        `http://localhost:8080/models/${modelId}/predict`,
        { inputs: input, shape: [1, 784] }
    );
    return response.data;
}
```
